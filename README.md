## Flagship Project

<a href="https://github.com/omobolajiadeyan/phishguard-ai">
  <img align="right" width="380" src="https://github-readme-stats.vercel.app/api/pin/?username=omobolajiadeyan&repo=phishguard-ai&theme=github_dark&hide_border=true"/>
</a>

**PhishGuard AI** is an offline, explainable phishing detection engine I created and maintain — zero external dependencies, pure Python stdlib.

Ships typosquatting detection (Levenshtein against 50 brand domains), HTTP redirect chain tracing, `.eml` email file analysis, and a **reusable GitHub Action** any CI pipeline can adopt with a single `uses:` line to scan URLs and upload SARIF findings to GitHub Code Scanning.
