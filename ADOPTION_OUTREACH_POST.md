# Adoption And Playtest Outreach Post

I am looking for practical feedback on two open-source security/software
projects I maintain.

PhishGuard AI is an explainable offline phishing-detection tool for URLs and
email. It is available as a GitHub Marketplace Action and supports JSON and
SARIF output for CI and code-scanning workflows.

Useful feedback:

- try the Marketplace Action on safe repository-owned URL fixtures;
- share whether the adoption guide is clear;
- report false positives or false negatives using public-safe samples;
- suggest documentation improvements for security teams.

Adoption guide:
https://github.com/omobolajiadeyan/phishguard-ai/blob/main/docs/ADOPTION.md

FreNiMi Checkers is a secure browser checkers project with local AI, PWA
support, and server-backed multiplayer/security documentation.

Useful feedback:

- test the public demo on desktop or mobile;
- check offline/PWA behavior;
- report accessibility or rules issues;
- review the secure session-revocation/demo documentation.

Playtest guide:
https://github.com/omobolajiadeyan/frenimi-checkers/blob/main/docs/PLAYTEST_GUIDE.md

I am especially interested in reproducible feedback: browser, operating
system, exact steps, expected result, actual result, and whether any issue
affects security, accessibility, or reliability.
